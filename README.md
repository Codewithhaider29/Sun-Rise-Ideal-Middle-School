## Overview
This is a free educational HTML template designed specifically for **Sun Rise Ideal Middle School**. The template is built using **Bootstrap 4.5.2**, ensuring a responsive and adaptable layout for all device types. Whether you're viewing on a desktop, tablet, or mobile, this template delivers an optimal experience.

## Features
- **Responsive design** for all screen sizes.
- Built with **Bootstrap 4.5.2** for customization and scalability.
- Modular design with multiple customizable sections and components.
- Integrated with useful **JavaScript libraries** such as:
  - **Isotope** (filtering & sorting)
  - **Owl Carousel** (sliders)
  - **Lightbox** (image galleries)
  - **Tabs** (tabbed navigation)
  - **Video** (embed videos)
- **Font Awesome icons** for scalable vector icons.
  
## Usage
1. **Download** the template files and **extract** them into a folder on your computer.
2. Open the `index.html` file in your browser to preview the template.
3. To customize:
   - Modify the **HTML**, **CSS**, and **JavaScript** files as needed.
4. Use the template as a foundation for your project or modify it to fit your school's branding.

## Customization
- **HTML**: Edit the `index.html` to change the structure and content of the template.
- **CSS**: Update `styles.css` to adjust colors, fonts, layouts, and overall design.
- **JavaScript**: Modify the `script.js` file to tweak interactive elements like carousels, modals, and animations.

## Dependencies
- **Bootstrap 4.5.2**
- **jQuery**
- **Isotope**
- **Owl Carousel**
- **Lightbox**
- **Tabs**
- **Video**
